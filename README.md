# CP & DSA Problemset for Newbies

A beginner-friendly, organized collection of competitive programming (CP) and data structures & algorithms (DSA) problems with notes and solutions.

## What you will find here
- Curated problems by platform (Codeforces, AtCoder, LeetCode, CodeChef)
- Clear file naming for quick search
- Problem notes (`.txt`) + solutions (`.cpp`)

## Repository structure
```
CP & DSA Problemset for Newbies/
├── Atcoder/
├── Codeforces/
├── Leetcode/
├── codechef/
└── README.md
```

## Naming convention
**Codeforces** files follow this pattern:
```
q{number}_cf_{rating}_{contestId}_{index}.txt
q{number}_cf_{rating}_{contestId}_{index}.cpp
```
Example:
- `q1_cf_800_2127_A.txt`
- `q1_cf_800_2127_A.cpp`



## Status
 New problems are added regularly.
